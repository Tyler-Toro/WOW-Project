# WOW-Project
Our game is called Escaping Toro Sanctum(ETS). It was worked on by Tyler Toro, Lavonte Carter, and Leo Redko.
It is a text-based adventure game that lets the user choose his own path. 



 



## Escaping Toro Sanctum
